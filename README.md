# CLI To-Do App

A beginner-friendly command line To-Do application built in Python to practice functions, loops, and error handling.

## Features

- Add new tasks
- View all tasks
- Delete tasks
- Mark tasks as complete
- Edit existing tasks
- Tasks are saved to a file (persistent storage)
- Input validation for safer user input
